# Identity Lifecycle Management Lab (Joiner-Mover-Leaver)

This project simulates real-world IAM lifecycle processes used in enterprises.

## Objective

Demonstrate Joiner, Mover, Leaver process using Microsoft Entra ID.

## Tools Used

* Microsoft Entra ID
