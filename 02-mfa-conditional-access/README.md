# Policy 01: MFA Enforced Conditional Access Entra
Company wants to enforce MFA for all users accessing any cloud application. I configured a Conditional Access policy to enforce MFA across all users and cloud applications, aligning with Zero Trust principles (Never Trust, Always Verify).

## Overview
Implemented Conditional Access policies to enforce secure authentication using MFA.

## Policy Implemented
- Require MFA for all users
- Applied to all cloud applications

## Key Concepts
- Conditional Access
- Zero Trust Security
- MFA Enforcement
